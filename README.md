# Python Battleship Game

A console-based Battleship game developed in Python where players attempt to locate and sink randomly generated ships using a limited number of shots.

## Features

- Randomized ship placement
- Multiple ship sizes
- Input validation
- Hit and miss detection
- Ship sinking detection
- Win and loss conditions
- Turn-based gameplay

## Concepts Used

- Python Programming
- Functions
- Lists and 2D Arrays
- Random Number Generation
- Input Validation
- Game Logic
- State Management

## Technologies

- Python
- Command Line Interface

## What I Learned

This project strengthened my understanding of program structure, game development fundamentals, input validation, and managing application state across multiple functions.
